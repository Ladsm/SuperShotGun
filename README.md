# Super Shotgun

## Info

Super Shotgun is a CLI static site generator made in C++.  
Please build your site from the bottom to the top.  
SSG is more of a boiler plate maker then a full on static site generator(I called it Super Shotgun becouse I found the pun funny).

## How to Build

must have:

- A C++ compiler
- CMake

```sh
mkdir build

cd build

cmake ..

cmake --build .
```
